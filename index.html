CSS Transitions

What are CSS transitions?

CSS Transitions are the way you define the transition between the two states of an element. For example when two states of an element changes through an event like click, hover, focus, and etc. we could specify a way of how we want it to look during the change from one state of an element to another.

What are some things I can transition?

CSS transition allows you to transition many properties, such as the change in color, making the element move, rotating the element, size, and etc. These are done through having two state of the element. For example .box-1 { size...; background-color: blue; transition-duration: 1000ms; transition-property: background-color; } .box-1:hover { background-color: red; } The "transition-property" determines which properties from one state you want to change to the other state with a specific timing function and a duration (More on this later). The "transition-duration" specifies the duration of the transition from a starting state to another.

Transition Timing Function

In CSS transitions, you can specify the acceleration of the transition and it'll lets you control the speed of the transition over the duration. With this there are some preset one such as linear (The transition accelerates at a constant speed), ease-in (The transition starts slow and accelerates towards the end), ease-out (The transition accelerates at the start and decelerates towards the end), and ease-in-out (Starts and ends slow but accelerates towards the middle). You can also define your own acceleration using cubic-bezier. This is done through the "transition-timing-function" property.
